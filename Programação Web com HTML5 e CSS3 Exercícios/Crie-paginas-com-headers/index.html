Como funciona a Web e o que é uma página web?

A Web é uma aplicação onde documentos e/ou páginas são interligadas através de links, nela tem o lado cliente e o lado servidor, quando um usuário digita um endereço no navegador ou clica em um link recebido, o navegador solicita ao servidor esse arquivo, o servidor então procura o arquivo e manda de volta pro cliente que recebe o conteúdo e renderiza. Uma página web é um documento escrito em HTML que é mostrado por um navegador.


Quais são os papéis das linguagens HTML e CSS em uma página web?

A linguagem HTML é uma linguagem de marcação que é usada para estruturar os elementos de uma página, como títulos, parágrafos e imagens. Já o CSS é uma linguagem de estilos que está relacionada com a estética dos elementos da página, como cores, tamanhos e posicionamento.


O que são ambientes de programação?

Também chamado de ambiente de desenvolvimento, o ambiente de programação é onde é realizado o trabalho de codificação e construção de um site.
